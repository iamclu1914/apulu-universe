# AI Short Film – Full Prompt Library (Higgsfield Blog)

Source: https://higgsfield.ai/blog/ai-short-film-youtube-guide

Clipped: 2026-04-07

---

Comprehensive guide for producing a cinematic short film entirely using AI. Covers character creation with Soul ID, location generation, scene-by-scene prompt breakdowns, and B-roll techniques. Uses Higgsfield Cinema Studio and Soul Cinema as primary tools, with Nano Banana Pro for reference sheets.

## Characters
- Adil (hero cop), Dave (partner), Selena (wife)
- Created via Higgsfield Soul Cinema with Soul ID for facial consistency
- Character reference sheets generated with Nano Banana Pro (turnaround views)

## Locations
- Locker room, police car interior, dark hallway
- Location reference sheets for architectural consistency
- Each location prompted once, then referenced throughout

## Scene Structure
- Scene 1: Locker Room — comedy banter, character establishment
- Scene 2: Patrol Car — dialogue-driven, dashcam aesthetic, radio interruption as tonal shift
- Scene 3: The Surprise — tactical entry fakeout into surprise party reveal

## Prompt Techniques
- Soul ID tags (@Character-Name) for character consistency
- Location tags (@Location-Name) for setting consistency
- Camera direction in prompts: handheld, static, dashcam, POV
- Shot duration specified (e.g., "7 seconds, handheld")
- Detailed aesthetic descriptors: "DVR/security camera aesthetic, cheap wide-angle lens distortion, mild compression artifacts"

## B-Roll
- Dashcam driving shots with police procedural aesthetic
- Radio close-ups for tension building
- Side window street views for grounding

## Tools Used
- Higgsfield Cinema Studio — dialogue scenes, character acting
- Higgsfield Soul Cinema — character generation, location generation, B-roll
- Nano Banana Pro — character and location reference sheets

## Key Advice
"Start with your characters and locations first — consistency there is what makes every scene feel like it belongs to the same film."
