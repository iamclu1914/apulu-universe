## Quick Start: The 5-Layer Formula

Kling 3.0 thinks like a director. Use this structure:

```
SCENE → CHARACTERS → ACTION → CAMERA → AUDIO & STYLE
```

### Example:
```
Dimly lit jazz club at midnight, warm amber lights through cigarette smoke.
The singer in a sequined gown stands at the microphone, eyes closed.

Timeline:
(0-4s): Stands still gathering emotion, camera holds medium
(4-9s): Opens eyes, begins singing soulfully, camera pushes in
(9-13s): Emotion builds, hand gestures, camera reaches close-up
(13-15s): Final note held, tear forms, camera holds on face

Camera: Slow dolly push-in from medium to intimate close-up

Audio: Sings with smoky voice, piano accompaniment, soft club ambience

Style: Shot on 35mm film, warm color grade, shallow depth of field

Negative: No text overlays, no face distortion, keep identity consistent
```

## Start & End Frame Strategy

### Start Frame Controls:
- Opening pose/emotion
- Camera position
- Lighting and shadows
- Color palette
- Aspect ratio

### End Frame Controls:
- Final pose/expression
- Closing emotional tone
- Where camera settles

### Best Practices:
1. ✓ Keep lighting consistent across both frames
2. ✓ Show emotional progression (calm → surprised, still → expressive)
3. ✓ Maintain subject identity
4. ✗ Avoid extreme perspective jumps

## Character Consistency (Critical!)

### Method 1: Elements Feature (Recommended)
1. Upload character reference image
2. Mark as "Element" in Kling 3.0
3. Reference in prompts: "The [Element] walking..."

### Method 2: Character Reference
- Use same reference for start AND end frames
- Generate end frame using start as reference

## Key Principles to Remember

1. **Think like a DIRECTOR** - Give production instructions, not descriptions
2. **Camera direction is ESSENTIAL** - Without it, Kling defaults to static
3. **Break action into TIMELINE steps** - Use (0-4s), (4-9s), (9-13s), (13-15s)
4. **Use NEGATIVE prompts aggressively** - Prevents common issues
5. **Character consistency = Elements feature** - Upload and reference your character

---

## Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| Face morphing | Add: "consistent face, same identity" |
| Background flicker | Add: "stable background, consistent lighting" |
| Camera ignored | Use ONE explicit camera move |
| Stuck at 99% | Add clear end-state description |
| Style drift | Repeat style twice in prompt |

## Workflow Summary

```
1. DEFINE CHARACTER
   ↓ Create reference image
   ↓ Upload to Elements
   
2. BUILD PROMPT (5 Layers)
   ↓ Scene → Character → Timeline → Camera → Audio
   
3. CREATE FRAME PAIR
   ↓ Start Frame (opening)
   ↓ End Frame (closing)
   ↓ Keep lighting consistent
   
4. GENERATE & REFINE
   ↓ Upload to Kling 3.0
   ↓ Test 5s first
   ↓ Extend to 15s
   ↓ Iterate 3-5 times
```
