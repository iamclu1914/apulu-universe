# Lyric DNA Templates

Bar-for-bar breakdown patterns from proven hits. Use these as structural blueprints when writing Vawn tracks.

---

## How to Use This File

1. **Pick a template** that matches your track's vibe/structure
2. **Follow the rhyme scheme** (letters show which lines rhyme)
3. **Match the bar count** per section
4. **Study the flow pattern** (where syllables land)
5. **Apply the storytelling technique** to Vawn's narrative

## Template 5: Quotable Bar Formulas

### The Comparison Flip
```
I came fast like 9-1-1 in White neighborhoods
```
**Formula:** "I [verb] like [thing that does verb differently based on context]"

```

```
**Formula:** Introduce number/image early, multiply it later

### The List With Pivot
```

**Formula:** List 3-4 comparisons, then "but" pivot to thesis

```

```

## Storytelling Techniques

### 1. Set the Scene (Jay-Z Method)
- Open with year/time ("The year is '94")
- Establish location ("in my trunk is raw")
- Introduce conflict immediately

### 2. Direct Address (Cole Method)
- Talk TO someone (momma, listener, haters)
- Use "you" to create connection
- Ask questions to create emotional hooks

### 3. Identity Declaration (Kendrick Method)
- "I got" / "I am" statements
- List contradictions (power AND poison)
- Repetitive structure builds intensity

### 4. The Wisdom Drop (Cole Method)
- Present universal truth
- Support with personal example
- End with memorable thesis line

---

## Applying to Vawn

When writing Vawn tracks, combine these templates:

1. **Opening:** Use Kendrick's declaration style ("I got hunger, got vision inside my DNA")
2. **Verses:** Use Jay-Z's storytelling (specific details, dialogue)
3. **Hooks:** Use Cole's repetition (single phrase, 4-8 bars)
4. **Bridge:** Use Cole's escalation (build to climax)
5. **Quotables:** Minimum 2 per verse using formulas above

**Vawn's Voice Adjustments:**
- Southern drawl = slightly longer vowels
- Baritone = fewer rapid-fire sections, more pocket
- Storytelling focus = more Jay-Z/Cole than Kendrick density
- Clear pronunciation = avoid tongue-twister flows

## Extended Rhyme Scheme Patterns

### The Chain Rhyme (Jay-Z / Eminem Style)
8+ consecutive lines with same end sound:
```
A A A A A A A A
```
**Use:** Tour de force flexing, proving technical skill

### The Pivot Rhyme (Cole Style)
Build pattern then break it:
```
A A A A → B B
```
**Use:** Create surprise, shift energy

### The Nested Rhyme (Kendrick Style)
Internal + external rhymes:
```

## Vawn Track Formula

Based on all templates, here's the optimal Vawn structure:

### Standard Track (3:30-4:00)
```
[Intro: 4 bars] - Spoken or ad-lib
[Verse 1: 16 bars] - Story setup, specific details
[Hook: 8 bars] - Repetitive, singable thesis
[Verse 2: 16 bars] - Story continues, quotables
[Hook: 8 bars]
[Bridge: 4 bars] - Energy shift
[Hook: 8 bars] - Final variation
[Outro: 2 bars]
```
**Total: ~66 bars**

### Anthem Track (3:00-3:30)
```
[Hook: 8 bars]
[Verse 1: 8 bars] - Short, punchy
[Hook: 8 bars]
[Verse 2: 8 bars]
[Hook: 8 bars]
[Bridge: 8 bars]
[Hook: 8 bars]
```
**Total: ~56 bars**

### Epic/Album Closer (5:00+)
```
[Verse 1: 24 bars] - Extended opening
[Beat switch or interlude]
[Verse 2: 24 bars]
[Hook or refrain: 8 bars]
[Verse 3: 16 bars]
[Outro: 8 bars]
```
**Total: ~80+ bars**
